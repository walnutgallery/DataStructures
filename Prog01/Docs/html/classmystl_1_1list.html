Detailed Description

template<typename T>
class mystl::list< T >

Doubly-linked list.

Template Parameters
TData type

Assumes the following: There is always enough memory for allocations (not a good assumption, just good enough for our purposes); Functions not well-defined on an empty container will exhibit undefined behavior, e.g., front().
